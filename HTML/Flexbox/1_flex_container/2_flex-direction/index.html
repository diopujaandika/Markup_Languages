Flex Direction

Flex Direction adalah properti dalam CSS Flexbox yang ditunakan untuk menentukan arah atau posisi susunan flex item di dalam sebuah flex container.

Value Flex Direction

  1. flex-direction: row
  2. flex-dirextion: row:reverse
  3. flex-direction: column
  4. flex-direction: column-reverse

flex-direction: row

flex-direction: row digunakan untuk menyusun flex items secara horizontal dari kiri ke kanan tanpa memperhatikan ukuran asli flex items. Jadi seberapa banyak flex itemsnya, flex items akan terus disusun dalam satu baris penuh.
