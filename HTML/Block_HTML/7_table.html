Table

Elemen table digunakan untuk membuat sebuah table. Didalamnya terdapat elemen tr atau table row yang digunakan untuk membuat baris baru. Dan elemen td atau table data digunakan untuk membuat wadah yang digunakan untuk menampng data dalam table. Dan elemen th atau table header digunakan untuk membuat sebuah judul data dalam table.
